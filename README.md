# Laguno Dashboard

Website e dashboard para o Laguno Discord Bot.

## Stack

- Next.js 14 + TypeScript
- Tailwind CSS
- Express API
- MongoDB

## Deploy

- Frontend: Vercel
- Backend: Railway (pasta `api/`)
